How to deploy hexo website and host on github

In order to deploy my hexo website I used two github respositories:

  1. dddotcom.github.io / [username].github.io

    • this repo will be used to host your deployed hexo website
    • as long as it has an index.html file, it will display that on the webpage

  2. dddotcom.github.io-hexo / [username].github.io-hexo

    • this repo will be used to version control the important files of your website so that it can be recovered if lost on your local machine
      the following files are of particular importance:
      • scaffolds/*.md
      • .gitignore
      • _config.yml
      • package.json
