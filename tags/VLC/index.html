Streaming Video from Raspberry Pi to Windows PC

What you’ll need

  • Raspberry Pi running the Raspbian OS with internet connection
  • RasPi NoIR Camera

For troubleshooting I find it helpful to hook up your Pi to a monitor, mouse, and keyboard.

  • Monitor
  • HDMI Cable
  • USB mouse
  • USB keyboard
    Note: Raspberry Pis have a limited amount of USB ports. From 1 (Model A) to 4 ports (Model B+). I used Model B, you can view the different model specs here and here

Connecting the camera

To connect the camera to the Pi, first make sure that it is powered off.

The camera connects to the Pi via the CSI port, which is located in between the Ethernet and HDMI ports.

To connect the camera, pull up the plastic bar on the CSI port and insert the camera’s ribbon cable into the port with the tin contacts facing the HDMI port. Make sure that it is evenly inserted. Afterwards, press down on the plastic bar to lock and secure the cable into the port.

Enabling the camera

Power on the Pi and run the following command to enter the Setup Options menu:

Click the down arrow button until you select the Enable camera option. Hit enter and select <Enable> when prompted Enable support for Raspberry Pi camera?. Then hit enter to return to the original menu.

Click the right arrow button until you select <Finish>. Hit enter and when prompted to reboot, select <Yes> and then hit enter again to reboot the Pi.

When the Pi is powering on, the camera’s red indicator light should turn on and you should see the view from the camera on your monitor for a few seconds before entering the Raspbian Desktop view.

Streaming video

Adapting Alex’s tutorial, I tried to stream video from the camera to my computer using VLC. In order to do so, you need to install VLC on both devices.

When installing VLC on your computer, make sure you have the latest version, or streaming might not work properly. Note: My PC is currently running VLC media player 2.1.5 Rincewind

Click on Media then Open Network Stream.
Then enter the URL of the Raspi’s video stream. This will be http://[Raspi IP]:[port]. Press play to view the stream.

The stream might be laggy, so adjust the -w(width), -h(height), and -fps(frames per second) parameter values to scope out the maximum quality and frame rate that your PC and network can handle.

Other notable parameters that you can change are the dst(port), which will change the stream’s output port, and the -t parameter. The t parameter is the time in milliseconds that the camera will run for. If it is set to 0, the camera will remain on and keep streaming until you exit with ctrl+c
